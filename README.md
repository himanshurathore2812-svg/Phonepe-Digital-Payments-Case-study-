# 📊 PhonePe Digital Payments Analysis

## Project Overview

This project analyzes PhonePe digital payment transactions and demographic data across Indian states and districts from Q1 2018 to Q2 2021. The objective is to uncover transaction trends, user behavior, device preferences, and demographic relationships to derive actionable business insights.

## Dataset

The analysis uses five datasets containing:

* State-level transaction and user information
* Transaction type breakdowns
* Device brand usage statistics
* District-level transaction and user data
* District demographic information

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Analysis Performed

* Conducted exploratory data analysis (EDA) to identify transaction growth trends across states and years.
* Analyzed the distribution of transaction types and app usage patterns.
* Identified top-performing states based on transaction volume and average transaction value (ATV).
* Examined smartphone brand preferences and calculated device usage ratios.
* Performed data quality checks by comparing district-level and state-level aggregates.
* Merged transaction and demographic datasets to study user penetration and population relationships.
* Investigated the correlation between population density and transaction volume.
* Created interactive visualizations including line charts, bar charts, scatter plots, and pie charts.

## Key Insights

* Digital transactions showed consistent growth between 2018 and 2021.
* A few states accounted for a significant share of overall transaction volume.
* Peer-to-peer payments emerged as the dominant transaction category.
* Population density exhibited a positive relationship with transaction activity.
* User penetration and app engagement varied considerably across states.
* Device brand preferences differed among regions, providing opportunities for targeted optimization.

## Business Recommendations

* Increase marketing efforts in low-adoption regions.
* Focus customer acquisition strategies on densely populated districts.
* Optimize the application for dominant smartphone brands.
* Develop personalized campaigns based on transaction patterns and user behavior.

## Project Outcome

This analysis provides a comprehensive understanding of PhonePe's digital payment ecosystem and demonstrates the use of Python for data cleaning, exploratory data analysis, data validation, statistical analysis, and visualization to support data-driven decision making.
